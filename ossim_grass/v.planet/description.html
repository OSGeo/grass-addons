DESCRIPTION

v.planet.py : Scripts to generate vrt layer from a selected grass vector layer and add/zoomTo/remove the data to ossimplanet using the ossimplanet tcp listner.

OPTIONS

[-a] : add map and zoom to it

[-r] : remove map

[map] : input layer

[host] : ossimplanet listner settings "host" (default to "loacalhost")

[pport] : ossimplanet listner settings "position message port number" (default to "7000")

[dport] : ossimplanet listner settings "data port number" (default to "8000")

[brush] : brush color - syntax : 'r,g,b'

[pen] : pen color - syntax : 'r,g,b'

[size] : point size - 'x,y'

[fill] : fill polygons - [0 / 1]

[thickness] : thickness - [0 / 1]

EXAMPLE

Add selected map to ossimplanet screen and zoom to it : 
v.planet.py map=elevation.10m -a

Remove selected map from ossimplanet screen : 

v.planet.py map=elevation.10m -r
