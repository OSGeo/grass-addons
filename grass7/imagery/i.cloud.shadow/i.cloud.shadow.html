DESCRIPTION

i.cloud.shadow detects cloud shadows among dark pixels in imagery data. It shifts cloud pixels based on sun zenith and azimuth for a range of possible altitudes of the clouds. For the best match of clouds and shadows (dark pixels) a classified output map is produced where value 1 represents clouds, 2 cloud shadows and 3 other dark pixels.

EXAMPLE

i.cloud.shadow dark_pixels=shadows clouds=clouds --o --v
