NAME

i.sentinel - Toolset for download and processing of Copernicus Sentinel products.

KEYWORDS

imagery, import, satellite, Sentinel

DESCRIPTION

The i.sentinel toolset consists of currently four modules:
i.sentinel.download
downloads Copernicus Sentinel products from Copernicus Open Access Hub
i.sentinel.import
imports already downloaded Sentinel products into GRASS GIS mapset
i.sentinel.preproc
imports and performs atmospheric correction on Sentinel-2 images
i.sentinel.mask
creates clouds and shadows masks for Sentinel-2 images

REQUIREMENTS

  • Sentinelsat library
  • Pandas library
