DESCRIPTION

r.in.srtm.region imports all SRTM V2.1 tiles covering the current region or region extents given with region into GRASS, patches the tiles together and optionally interpolates holes.

r.in.srtm.region downloads SRTM tiles from:
http://dds.cr.usgs.gov/srtm/
or optionally uses a local folder with previously downloaded files if the local option is given.

NOTES

r.in.srtm.region imports only SRTM tiles with 3 arcsec resolution.
