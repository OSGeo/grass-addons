DESCRIPTION

r.in.srtm.region imports all SRTM tiles covering the current region or region extents given with region into GRASS, patches the tiles together and optionally interpolates holes for SRTM V2.1.

r.in.srtm.region downloads SRTM V2.1 tiles from:
http://dds.cr.usgs.gov/srtm/
and SRTM V3 tiles from: http://e4ftl01.cr.usgs.gov/SRTM/SRTMGL3.003/2000.02.11/
or optionally uses a local folder with previously downloaded files if the local option is given.

NOTES

r.in.srtm.region imports only SRTM tiles with 3 arcsec resolution.
