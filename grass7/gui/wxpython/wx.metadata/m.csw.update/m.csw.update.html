DESCRIPTION

m.csw.update update csw connections resource xml file (required by the g.gui.cswbrowser and g.gui.metadata module). If -p flag is used only printing instead of writing is done.

The module also allows validate conections resources xml file agains xsd schema, remove not valid and not active csw connnections resources candicates.

NOTES

For dependencies and installation instructions see wiki page.

Writed new connections resources candidates are only active and valid.

Writing new connections resources candicates

Default source of the new candicates is spreadsheets file (*.ods), which is located in the module config/ directory. It is possible to use the updated document directly from the web address, see -l flag.

EXAMPLES

Write new resources connections candicates (default): 
m.csw.update url=API-cases.ods

Write new resources connections candicates with printing summary info: 

m.csw.update -s

Print only all new connections resources candicates (with following format '{Country}, {Govermental level}, {Api provider}: {Url}') with summary info: 

m.csw.update -ps

Print only active new connections resources candicates with summary info: 

m.csw.update -pas

Validate default connnections resources xml file against xsd schema plus validate individual csw connection resources (remove and print non active csw): 

m.csw.update -xc
