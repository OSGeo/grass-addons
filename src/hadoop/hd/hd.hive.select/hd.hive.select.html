DESCRIPTION

hd.hive.select module allows to query table of Hive.

NOTES

EXAMPLES

Below is example of HQL query with redirecting output to file 
hd.hive.select driver=hiveserver2 hql='SELECT linkid from mwrecord' out='tmp/linkid.hql'
