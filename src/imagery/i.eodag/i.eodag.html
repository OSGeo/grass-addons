NAME

i.eodag.py - Eodag interface to install imagery datasets from various providers.

KEYWORDS

eodag, sentinel, landsat, modis, datasets, download

SYNOPSIS

i.eodag.py
i.eodag.py --help
i.eodag.py [-l] dataset=string [provider=string] start=string end=string [timeout=integer] [--help] [--verbose] [--quiet] [--ui]

Flags:

-l
List the search result without downloading
--help
Print usage summary
--verbose
Verbose module output
--quiet
Quiet module output
--ui
Force launching GUI dialog

Parameters:

dataset=string [required]
Imagery dataset to search for
provider=string
Provider to use for searching/downloading, if none is provided the searching will be done according to the config file priority
start=string [required]
Start date ('YYYY-MM-DD')
end=string [required]
End date ('YYYY-MM-DD')
timeout=integer
Download timeout in seconds
Default: 300
