DESCRIPTION

r.soiltex2prop calculates the soil porosity (volume fraction), saturated hydraulic conductivity (cm/h) and wetting front suction (cm ; Green-Ampt) after Rawls et al, 1990. It takes input of soil texture proportions (sand, clay) in range [0.0-100.0].
