NAME

r.in.ogc imports raster data from an OGC API server.

KEYWORDS

vector, import topic

DESCRIPTION

The r.in.ogc suite is able to import raster data from several OGC API standards. It supports
  • OGC API Coverages - import raster data from an OGC API Coverages standard 
g.extension r.in.ogc

NOTES

r.in.ogc requires owslib Python library.
