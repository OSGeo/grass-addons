DESCRIPTION

The r.mcda.output module will export raster maps as a *isf file readable in jMAF, JAMM or 4eMka2 sofware Laboratory of Intelligent Decision Support Systems - Poznan University.

(GRASS python Script)

NOTES

This module, as all GRASS raster modules, will export cells based on the current region settings. See the g.region module for details.
