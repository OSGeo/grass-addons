DESCRIPTION

r.clim calculates solar radiation and percentage relative humidity maps. The calculation of the two values for each raster cell is based on the C code cclimm.c a simplified version of mtclim43.c by Peter Thornton (2000) of the NTSG, School of Forestry University of Montana. The cclimm.c code is thought for monthly data and doesn't consider arid correction so this module isn't indic ated for arid region.

The input units are:

  • Temperatures: degrees C
  • Precipitation: mm / day
  • Elevation: m
  • Slope: degrees
  • Aspect: degrees
  • Latitude: decimal degrees
  • E/W horizons: decimal degrees
while the output units are:
  • Relative humidity: %
  • Radiation: W/m2, average over daylight period
