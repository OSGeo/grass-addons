DESCRIPTION

v.out.ply exports vector point clouds from GRASS into a PLY file (PLY format).
If color definitions are present for each point, they will be exported as well.

EXAMPLE

v.out.ply archaeo_pointcloud

SEE ALSO

v.out.ascii, v.out.ascii.db (GRASS extension)
