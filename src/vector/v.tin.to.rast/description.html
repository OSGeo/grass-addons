NAME

v.tin.to.rast - Converts (rasterize) a TIN map into a raster map

KEYWORDS

TIN, vector, raster, conversion

SYNOPSIS

v.tin.to.rast
v.tin.to.rast help
v.tin.to.rast input=tin output=raster [--overwrite] [--verbose] [--quiet]

Flags:

--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

input=tin
Name of input TIN map
output=raster
Name of output raster map
