DESCRIPTION

v.in.ply imports point clouds stored as PLY file (PLY format) as vector points into GRASS.
If color definitions are present for each point, a corresponding GRASSRGB (RRR:GGG:BBB, varchar (11)) column is created and populated.

EXAMPLE

v.in.ply archaeo_pointcloud.ply
