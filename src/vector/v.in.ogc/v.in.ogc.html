NAME

v.in.ogc imports vector data from an OGC API server.

KEYWORDS

vector, import topic

DESCRIPTION

The v.in.ogc suite is able to import vector data from several OGC API standards. It supports
  • OGC API Features - import vector data from a OGC API Features standard 
g.extension v.in.ogc

NOTES

v.in.ogc requires owslib Python library.
