DESCRIPTION

g.xremove removes data files matching a pattern given by wildcards or POSIX Extended Regular Expressions. If the -f force flag is not given then nothing is removed, instead the list of selected file names is printed to stdout as a preview of the files to be deleted.

EXAMPLE

Delete all raster maps starting with "tmp_" in the current mapset: 
  g.xremove -f "tmp_*"
