DESCRIPTION

t.rast.climatologies is a module to calculate climatologies for singol days or months in a time series creating as output a new relative space time raster dataset

EXAMPLE

Daily series

Starting from a space time raster dataset it create a new space time raster dataset containing the daily maps of average for a multi year dataset 
        t.rast.climatologies strds=myinput output=dailyoutput

Monthly series

Starting from a space time raster dataset it create two new space time raster datasets containing the monthly maps of minimum and maximum for a multi year dataset 
        t.rast.climatologies strds=myinput granularity=month method=min,max output=monthlyoutputmin,monthlyoutputmax
