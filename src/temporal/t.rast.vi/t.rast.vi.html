DESCRIPTION

This module calculates vegetation indices from several space time raster datasets, using i.vi module.
It can apply cloud and shadow masks stored as space time datasets (raster or vector). The output space time raster dataset can be a new dataset or an existing one, in this case the new created maps will be added to the existing space time raster dataset.
