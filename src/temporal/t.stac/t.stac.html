DESCRIPTION

Spatial Temporal Asset Catalog (STAC) toolset for GRASS GIS allows the user to ingest and export STAC items, collections, and catalogs. The toolset is based on the PySTAC library and provides a set of modules for working with STAC APIs. STAC is a specification for organizing geospatial information in a way that is interoperable across software and data services. pystac-client is used to interact with STAC APIs.

t.stac.catalog t.stac.collection t.stac.item (WIP) t.stac.export

DEPENDENCIES

  • pystac (1.9.x)
  • pystac_client (0.7.x)
  • tqdm (4.66.x)

After dependencies are fulfilled, modules can be installed in GRASS GIS >= 7.8 using the g.extension module: 

g.extension extension=stac
