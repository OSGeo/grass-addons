DESCRIPTION

r.planet.py

- Generate vrt layer from a selected grass raster layer and add/zoomTo/remove the data from/to ossimplanet using the ossimplanet tcp listner.

- Generate ossimplanet 3d tiles from elevation data.

As default the module uses : host = 'localhost', pport = 7000, dport = 8000

The module will generate VRT data inside the directory : $MAPSET/vrt/raster and $MAPSET/vrt/raster/elevetion.

Using the flag "-d" if non name is selected (parameter : elev) the module will use the map's name.

Flags:

-a : Add selected map to Ossimplanet

-r : Remove selected map to Ossimplanet

-d : Generate 3d Tiles

-m : Use Open MPI

Parameters

:

map : Input map

Host : Ossimplanet listener IP

dport : Ossimplanet listener port to receive position messages

pport : Ossimplanet listener port to receive datasource

np : Number of MPI processors

tile : tiles dimension for 3d data

elev : Name for the exporeted elevation

EXAMPLE

Add selected map to ossimplanet screen and zoom to it : 
r.planet.py map=elevation.10m -a

Remove selected map from ossimplanet screen : 

r.planet.py map=elevation.10m -r

Generate 3D tiles : 

r.planet.py map=elevation.10m -d tile=3601
