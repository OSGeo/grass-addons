NAME

DESCRIPTION

This program prints a report of cell counts in the current region and under a sample of cells with positions specified by vector objects. It can also calculate Kvamme's gain factor which is a popular index in archaeological predictive modelling.

Notes

This program was originally developed as part of the GRASS 5 DST Predictive Modelling Toolkit.
