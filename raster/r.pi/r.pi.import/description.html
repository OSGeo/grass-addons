DESCRIPTION

The r.pi.import module imports values and assigns them to corresponding patches using a previously exported raster by r.pi.export. The raster must have the same extent, spatial resolution etc. in order to detect the same patches and assign the same patch IDs for all patches.

The program will be run non-interactively if the user specifies program arguments (see OPTIONS) on the command line. Alternately, the user can simply type r.pi.import on the command line, without program arguments. In this case, the user will be prompted for flag settings and parameter values.

NOTES

For export and import of patch values after e.g. R calculation r.pi.export and r.pi.import can be applied using the same raster.
