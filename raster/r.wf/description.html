DESCRIPTION

r.wf outputs the Width function from the outlet distance map (r.stream.distance output) and print out percentiles.

NOTES

The width function defined as the number of pixels having equal metric distance from the closing section measured by following the directions of drainage. They are calculated as a synthesis of the first 4 statistical moments (mean, variance, skewness, kurtosis) and percentiles, or the percentage of pixels within a certain distance from section closure (5%, 15%, 30%, 40%, 50%, 60%, 70%, 85%, 95%).

EXAMPLE

r.wf.py map=dst2out image=path/to/output/prefix
